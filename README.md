# Daily Coding Challenge

Welcome to my **Daily Coding Challenge** repository! This is a personal project where I commit to solving coding problems every day to build consistency and improve my problem-solving skills across various computer science topics.

## Purpose

The goal of this repository is to maintain a daily coding habit. Each day, I tackle problems from different domains to ensure comprehensive coverage and continuous learning. Consistency is key—aiming for daily commits and solutions.

## Topics Covered

I aim to cover a wide range of algorithmic and data structure topics, including but not limited to:

- **Arrays**
- **Strings**
- **Two Pointers**
- **Sliding Window**
- **Prefix Sum**
- **Hash Maps**
- **Stacks**
- **Queues**
- **Linked Lists (LL)**
- **Trees**
- **Graphs**
- **Searching**
- **Sorting**
- **Heaps**
- **Priority Queues**

Each problem will be solved in Python (and possibly other languages in the future), with clear explanations and code comments.

## Repository Structure

- Problems are organized by topic in separate files/folders.
- Each solution includes:
  - Problem statement (briefly)
  - Code implementation

## Getting Started

Feel free to explore the solutions, suggest improvements, or use them as references for your own learning. If you have similar goals, let's connect and motivate each other!
